# Pixel Gallery Pro

## Описание
Профессиональный просмотрщик изображений с поддержкой пиксельной графики, SVG, GIF, фильтрами и полноэкранным режимом.

## Возможности
- Загрузка папки с изображениями (PNG, JPG, GIF, SVG)
- Два режима просмотра: сетка / список
- Масштабирование и панорамирование
- Редактирование: поворот, фильтры, яркость, контраст
- Полноэкранный режим
- Сохранение настроек в браузере
